Kadena SpireKey leverages WebAuthn standards to deliver a secure backend, enabling end users to generate and store keypairs directly on their hardware devices. Developers can generate keypairs using Chainweaver V3 Alpha, integrate their application with Kadena SpireKey using the SDK package, or explore the technical documentation outlined in the KIPs.
